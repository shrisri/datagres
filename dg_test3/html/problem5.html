Program 4

Will write a master script, to create many subscripts, each of which will take sections of the file F1 and verify only those ranges. This will do the verification. If one of the child script fails, you have to report, which script failed, for what reason etc.

Instructions: To compile only this program type 'make Program4'. To run, type './Master.sh 4'. After running this program,it will automatically run the four other programs Program4sub1.sh,Program4sub2.sh,Program4sub3.sh,Program4sub4.sh for different sections of newfile2.txt and enter the results to a file called results.txt. To compile program 1-4 together, type 'make'
