Networking Commands:

1.ifconfig
This command is used to check IP address assigned to the system and to further configure network interface parameters.
Example: ifconfig -a

2.nslookup
nslookup is a program to query Internet domain name servers.
Example: nslookup www.google.com

3.netstat
Netstat command allows you a simple way to review each of your network connections and open sockets.
Example: netstat
