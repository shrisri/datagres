Program 4

Will write a master script, to create many subscripts, each of which will take sections of the file F1 and verify only those ranges. This will do the verification. If one of the child script fails, you have to report, which script failed, for what reason etc.

Instructions: To run only this program open dg_test1 directory and type 'make test4'.It will automatically run test4.sh and the four other programs test4sub1.sh,test4sub2.sh,test4sub3.sh,test4sub4.sh for different sections of newfile2.txt and enter the results to a file called results.txt. To run program 1-4 together, type 'make'
