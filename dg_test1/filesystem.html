Filesystem Commands:

1.mv
Used to move files from one directory to another.
Example: mv file1.txt d1
This will move file1.txt from current directory to directory d1

2.cat
Used to see the contents of a file.
Example: cat file1.txt
This will print all the contents of file1.txt

3.cd
Used to enter a directory.
Example: cd d1
This will enter directory d1

4.cp
Used to copy content of directory or file to another.
Example: cp file1 file2
This will copy contents of file1 to file2

5.ls
Used to see files or directory in a directory.
Example: ls d1
This command would display the contents of d1 directory

6.mkdir
Used to create a new directory.
Example: mkdir d1
This would create a new directory d1 in the current directory

7.rmdir
Used to delete a directory.
Example: rmdir d1
This would completely delete directory d1 from current directory.

8.pwd
Used to display the current path.
Example: pwd

9.rm
Used to remove a file.
Example: rm file1

10.touch
Used to create a file.
Example: touch file1

11.chmod
Used to change the permissions to a file.
Example: chmod 777 file1

12.find
Used to search for a specific file or directory.
Example: file d1 f1

13.dd
Used to convert and copy files.
Example: dd if=f1.txt of=f2.txt

14.wc
Stands for Word Count, implying that it is mainly used for counting letters,words,lines etc.
Example: wc -l file1
